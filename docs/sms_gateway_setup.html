SMS Gateway Configuration Guide

1. Accessing SMS Gateway Settings

  1. Log in to your admin panel
  2. Go to Settings
  3. Scroll down to find "SMS Notification" section
  4. Click to expand the section

2. Configuring SMS Gateways

A. Blessed Texts Setup

  1. Click on "BlessedText SMS Gateway" in the side menu
  2. Enter your Blessed Texts API Key
  3. Enter your Blessed Texts Sender ID
  4. Click "Save Changes"

B. Talk Sasa Setup

  1. Click on "TalkSasa SMS Gateway" in the side menu
  2. Enter your Talk Sasa API Token
  3. Enter your Talk Sasa Sender ID
  4. Click "Save Changes"

3. Switching Between Gateways

  1. Go to Settings > SMS Notification
  2. Look for "SMS Gateway" dropdown
  3. Select your preferred gateway:
    • Choose "Blessed Texts" to use Blessed Texts gateway
    • Choose "Talk Sasa" to use Talk Sasa gateway
  4. Click "Save Changes" at the bottom of the section

4. Testing the Configuration

  1. After selecting and saving your preferred gateway:
    • In the SMS Notification section, look for "Test SMS" button
    • Click the button to send a test message
    • Check the response to confirm it's working

5. Troubleshooting

If messages are not being sent through the selected gateway:

  1. Verify your API credentials are correct
  2. Ensure you've clicked "Save Changes" after selecting the gateway
  3. Clear your browser cache or do a hard refresh (Ctrl+F5)
  4. Check the SMS logs in the respective gateway's menu item
Note: Only one gateway can be active at a time. The system will use the selected gateway for all SMS communications.

For technical support or questions, please contact your system administrator.
